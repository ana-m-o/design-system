# Spin

A spinner for displaying loading state of a page or a section.

Read [Ant Documentation]([https://ant.design/components/spin/) for props and uses

The Spin usually wraps the content like `<Spin>Content of the page</Spin>`

By default the layout is horizontal for an empty `<Spin />`. In order to use a vertical layout use `className="uiv-vertical-spin"`

export const Template = (args) => <Spin {...args}></Spin>;
